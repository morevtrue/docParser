# Agent Pack — PUBLIC API (DocParser)

## API-контракт

### POST /api/process
- Принимает: `multipart/form-data` с DOCX-файлами (входные документы + шаблоны).
- Возвращает: ZIP-архив (`application/zip`) с заполненными DOCX + `report.json`.
- Ошибки:
  - 400 — не DOCX файл или файлы не загружены
  - 422 — не удалось распарсить документ
  - 500 — внутренняя ошибка сервера

### GET /api/health
- Возвращает: `{"status": "ok"}`

## Принципы

- Все запросы от фронтенда идут через `/api/*`.
- Vite dev proxy проксирует `/api/*` на бэкенд (порт 8000).
- Frontend не обращается к бэкенду напрямую по порту.
