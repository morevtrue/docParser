# Agent Pack — START (DocParser)

Этот набор файлов предназначен для ИИ-агента, работающего над проектом DocParser.

## Политика чтения

**По умолчанию агент читает только:**

1. `agent/*` (этот набор)
2. `phases/*` (по фазам: что делать / что сделано / тесты)

**Все остальные документы** (включая `sources-of-truth/*`) — **только по запросу**,
когда это реально нужно для конкретной задачи.

## Что читать дальше

1. `agent/INVARIANTS.md` — ограничения и запреты
2. `agent/PUBLIC_API.md` — API-контракт
3. `agent/TESTING.md` — как проверять изменения

Если задача привязана к конкретной фазе — переходи в `phases/phase-X/WHAT_TO_DO.md`.

## Принцип работы с фазами

Каждая фаза содержит три файла:
- `WHAT_TO_DO.md` — что нужно сделать
- `DONE.md` — что уже сделано (заполняется по завершении)
- `TESTS.md` — как проверить результат

Агент работает с текущей активной фазой. Завершённые фазы — только для справки.
