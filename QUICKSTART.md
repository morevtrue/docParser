# Запуск DocParser с нуля

## Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (включает Docker + Compose)

Больше ничего устанавливать не нужно — Python, Node.js и все зависимости собираются внутри контейнеров.

## Запуск

```bash
docker compose up -d --build
```

Первый раз займёт 2–5 минут (скачивает образы, устанавливает зависимости). Последующие запуски без `--build` — секунды.

## Адреса после запуска

- Приложение: http://localhost:5173
- API (Swagger): http://localhost:8000/docs

## Остановка

```bash
# Ctrl+C в терминале, или в другом окне:
docker compose down
```

## Как пользоваться

1. Открыть http://localhost:5173
2. Загрузить документы в три зоны (только `.docx`):
   - **Документы от заказчика** — Запрос ТКП
   - **Документы поставщика** — Карточка поставщика + Коммерческие условия
   - **Шаблоны** — пустые шаблоны для заполнения
3. Нажать **Обработать** → скачать ZIP с заполненными документами

Тестовые файлы лежат в папках `Основной набор/` и `Шаблоны/` в корне проекта.
